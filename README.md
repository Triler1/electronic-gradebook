# Electronic Gradebook

Лабораторная работа: **Электронный журнал группы**.

## Задача

Необходимо реализовать простой электронный журнал.

Система хранит студентов и их баллы.
Для каждого студента нужно быстро получать текущий результат по имени или по id.

## Обязательная часть

Реализовано:

- добавление студента в журнал;
- выставление или обновление балла;
- получение балла по имени;
- получение балла по id;
- вывод всех студентов и их результатов.

## Вариативная часть

Реализован пункт **№6**:

- удаление студента из журнала.

## Для упорных

Реализовано:

- тесты для основных функций;
- разбиение проекта на несколько файлов.
